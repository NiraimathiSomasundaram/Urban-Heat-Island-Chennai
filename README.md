# Urban-Heat-Island-Chennai
Urban Heat Island analysis using Landsat-8 LST and Google Earth Engine

## Project Code: P2
## India Space Academy — Summer Internship 2026

## Objective
To map Urban Heat Island zones in Chennai using 
Landsat-8 Land Surface Temperature data on Google Earth Engine.

## Study Area
Chennai, Tamil Nadu, India

## Key Results
- Minimum LST: 27.81°C
- Maximum LST: 54.43°C
- UHI Area (>35°C): 1480.37 sq km

## Tools Used
- Google Earth Engine (GEE)
- Landsat-8 OLI/TIRS
- WPS Writer
